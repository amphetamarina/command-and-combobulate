# Command & Clanker sprite assets

Custom-generated sprite art for Command & Clanker, drawn to match the
file structure and per-sprite canvas sizes of the [acdrnx Sci-Fi Strategy
Mech Buildings pack](https://opengameart.org/content/sci-fi-strategy-mech-buildings-isometric-asset-pack)
(CC0). Sharing the structure lets the renderer use the same sprite
keys and tile anchoring code unchanged.

## Contents (v1)

- `sci-fi/buildings/Step {1,2,3}/{foerderturm,kraftwerk,raffinerie,treibhaus}.png`
  Four building silhouettes, three colour variants each.
- `sci-fi/units/Mech/step {1,2,3}/Idle/idlesued.png`
  Three south-facing idle mech variants.

## Canvas sizes (do not change without updating anchoring)

| Sprite | px |
|---|---|
| foerderturm | 62x131 |
| kraftwerk | 63x46 |
| raffinerie | 64x83 |
| treibhaus | 64x51 |
| mech idlesued | 29x32 |

The renderer anchors every building at (0.5, 1.0) and positions it
at the south corner of its tile, so the bottom-centre pixel of each
PNG must coincide with the south point of the iso diamond base. If
new art is dropped in at different dimensions, the anchor or
position may need adjustment.
